👉# 🚀 Machine Learning Web App (Flask + Render Deployment)

A simple end-to-end **Machine Learning web application** built using Flask and deployed on Render.  
The app takes user input and provides predictions using a trained ML model.

---

## 🌐 Live Demo
👉 https://house-price-prediction-1-j48t.onrender.com/

---

## 📌 Project Overview
This project demonstrates how a machine learning model can be integrated into a web application using Flask and deployed on a cloud platform (Render).

Users can:
- Input data through a web form
- Get real-time predictions
- Experience ML model deployment in production

---

## 🛠 Tech Stack

- Python 🐍
- Flask 🌐
- Scikit-learn 🤖
- Pandas 📊
- NumPy 🔢
- Gunicorn 🚀
- HTML/CSS (Frontend)
- Render (Deployment)
